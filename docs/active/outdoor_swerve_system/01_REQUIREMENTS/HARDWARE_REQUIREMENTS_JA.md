## 1. はじめに

### 1.1 目的

本文書は、屋外優先車椅子輸送ロボットの**すべてのハードウェア要件**を統合します。これには、コンピュートプラットフォーム、センサー、アクチュエーター、電源システム、機械構造、環境保護が含まれます。

### 1.2 スコープ

本仕様は以下をカバーします:
- **コンピュートプラットフォーム** - GMKtec Nucbox K6仕様
- **センサー** - 3D LiDAR、カメラ、IMU、エンコーダー、安全センサー
- **アクチュエーター** - スワーブドライブモーター、ブレーキ、ドッキング機構
- **電源システム** - バッテリー、BMS、配電、充電
- **機械構造** - シャーシ、車椅子取り付け、防水
- **環境保護** - IP等級、温度、防水

## 2. コンピュートプラットフォーム要件

### 2.1 メインコンピューター（HW-COMP-MAIN）

| 要件ID | 要件 | 優先度 | 検証方法 |
|--------|------|--------|----------|
| HW-COMP-MAIN-001 | システムはメインコンピュートプラットフォームとしてGMKtec Nucbox K6を使用しなければならない | 必須 | ハードウェアBOM |
| HW-COMP-MAIN-002 | メインコンピューターはAMD Ryzen 7 7840HS CPU（8コア、16スレッド、3.8-5.1 GHz）を持たなければならない | 必須 | データシート |
| HW-COMP-MAIN-003 | メインコンピューターは32GB DDR5 5600MHz RAMを持たなければならない | 必須 | データシート |
| HW-COMP-MAIN-004 | メインコンピューターはAMD Radeon 780M統合GPUを持たなければならない | 必須 | データシート |
| HW-COMP-MAIN-005 | メインコンピューターは1TB PCIe 4.0 SSDストレージを持たなければならない | 必須 | データシート |
| HW-COMP-MAIN-006 | メインコンピューターは≥4× USB 3.0ポートを持たなければならない | 高 | データシート |
| HW-COMP-MAIN-007 | メインコンピューターは≥1×ギガビットイーサネットポートを持たなければならない | 必須 | データシート |
| HW-COMP-MAIN-008 | メインコンピューターはUbuntu 22.04 LTSをサポートしなければならない | 必須 | 互換性テスト |
| HW-COMP-MAIN-009 | メインコンピューターは-10°C～+45°Cで動作しなければならない（冷却付き） | 高 | 熱テスト |
| HW-COMP-MAIN-010 | メインコンピューターはIP54+エンクロージャーに収納されなければならない | 必須 | 環境テスト |

**合計要件: 10**
