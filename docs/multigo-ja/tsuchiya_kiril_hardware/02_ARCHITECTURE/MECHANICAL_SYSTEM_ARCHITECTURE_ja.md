## シャーシ構造

**フレーム材料:** アルミニウム6061-T6押出材（40mm × 40mm）
- 軽量: 15kgフレーム重量
- 高強度対重量比
- 耐腐食性（陽極酸化仕上げ）
- モジュラー設計（ボルト組み立て）

**寸法:**
- 長さ: 1200mm
- 幅: 900mm
- 高さ: 600mm（車椅子なし）
- 地上高: 150mm
- ホイールベース: 800mm
- トレッド幅: 700mm

**重量配分:**
- 前部: 45%（センサー、コンピュートユニット）
- 後部: 55%（バッテリー、充電ポート）
- 重心: 高さ350mm、横方向中央
- 総重量: 150kg（バッテリー含む）

## スワーブドライブモジュール（×4）

```
     ┌─────────────┐
     │ステアモーター│ (12V、50W、エンコーダー)
     └──────┬──────┘
            │ ベルトドライブ（3:1減速）
            ▼
     ┌─────────────┐
     │スイベルプレート│ (ベアリング取り付け)
     └──────┬──────┘
            │
            ▼
     ┌─────────────┐

     │  + ホイール  │ (6.5インチ直径)
```

**モジュールごとの仕様:**
- ドライブモーター: 200W、48V、ホール効果エンコーダー
- ステアモーター: 50W、12V、絶対エンコーダー
- ホイール直径: 165mm（6.5インチ）
- タイヤ: 空気圧、屋外トレッドパターン
- スイベルベアリング: 深溝玉軸受、内径30mm
- ステアリング範囲: 360°連続回転
- 最大トルク（ドライブ）: 15 Nm
- 最大速度: 2 m/s（無負荷）

## 車椅子ドッキング機構

### 機械設計

```
側面図（ロボット後部）
┌──────────────────────────────────┐
│  ロボットシャーシ                 │
│                                  │
│         ┌────────────┐           │
│         │  ドッキング │           │
│         │   アーム    │◄──────────┼─ サーボアクチュエーター
│         │  (折り畳み) │           │
│         └────────────┘           │
│              │                   │
│              ▼                   │
│         [係合                    │
│          機構]                   │
│              │                   │
└──────────────┼───────────────────┘
               │
        車椅子キャスタースロット
```

**コンポーネント:**
- 2×折り畳みアーム（アルミニウム、サーボ駆動）
- 係合機構: スプリング式ピン
- アライメントガイド: テーパー付きファンネル（±50mm公差）
- センサー: ホール効果スイッチ（係合確認）
- ArUcoマーカー: 100mm × 100mm、高さ500mmに取り付け

**ドッキングシーケンス:**
1. ビジュアルアライメント（ArUcoマーカー検出）
2. アーム展開（サーボ駆動、2秒）
3. ロボットが係合位置まで後退
4. スプリングピンが車椅子キャスタースロットに係合
5. ホール効果センサーで確認

## センサー取り付け

### LiDAR（2D、前面取り付け）
- モデル: SICK TiM571または同等品
- 取り付け高さ: 400mm
- 取り付け角度: 0°（水平）
- 視野: 270°
- 振動絶縁: ゴムダンパー

### カメラ（×3）
- 前面カメラ: 1920×1080、90° FOV、高さ1200mm
- 後面カメラ: 1280×720、120° FOV（魚眼）、高さ800mm
- ドッキングカメラ: 1280×720、60° FOV、高さ500mm（ArUco検出）
- 取り付け: 調整可能ブラケット、IP65エンクロージャー

### IMU
- 取り付け: シャーシ中心、剛性ボルト留め
- 方向: ロボット座標フレームと揃える

## エンクロージャー

### コンピュートユニットエンクロージャー
- 材料: 粉体塗装鋼、IP54
- 寸法: 400mm × 300mm × 200mm
- 取り付け: 振動絶縁（ゴムグロメット）
- 冷却: 強制空気（40mmファン）、フィルター付き吸気
- アクセス: ヒンジドア、クォーターターンファスナー

### バッテリーエンクロージャー
- 材料: アルミニウム、IP65
- 寸法: 500mm × 350mm × 150mm
- 取り付け: 中央取り付け、低CoG
- 冷却: パッシブ（フィン付きアルミニウム）
- アクセス: 工具不要ラッチ（クイックバッテリー交換）
- 火災抑制: サーマルヒューズ（85°C）

### 外部ディスプレイエンクロージャー
- 取り付け: 側面パネル、高さ1200mm
- 材料: ポリカーボネートシールド（3mm）、IP65ガスケット
- 角度: 15°上向き傾斜（人間工学的視認性）

## サスペンションとコンプライアンス

- アクティブサスペンションなし（剛性シャーシ）
- 空気圧タイヤによるコンプライアンス（40 PSI）
- スワーブモジュールがある程度のコンプライアンスを提供（ゴムブッシング）
- 地上高: 150mm（ほとんどの障害物を回避）

## 防水性

- すべてのエンクロージャー: 最低IP54（クリティカルコンポーネントはIP65）
- ケーブル侵入: IP定格ケーブルグランド
- ベント: Gore-Texメンブレンベント（圧力均等化、湿気バリア）
- シール: シリコンガスケット、EPDMゴム
- 排水: エンクロージャー内の傾斜床、低点に排水穴
